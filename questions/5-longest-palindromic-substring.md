## Problem Statement

"Given a string `s`, return *the longest palindromic substring* in `s`."

**Examples:**
- Input: `s = "babad"`, Output: `"bab"` (Note: `"aba"` is also valid)
- Input: `s = "cbbd"`, Output: `"bb"`
- Input: `s = "a"`, Output: `"a"`
- Input: `s = "ac"`, Output: `"a"`

**Constraints:**
- `1 <= s.length <= 1000`
- String consists of digits and English letters only
