## Problem Description

"Given an array `nums` of _n_ integers and an integer `target`, are there elements _a_, _b_, _c_, and _d_ in `nums` such that _a_ + _b_ + _c_ + _d_ = `target`?" The task is to find all unique quadruplets that sum to the target value.

**Key Requirements:**
- "The solution set must not contain duplicate quadruplets"
- Return all valid four-element combinations

**Examples:**

**Example 1:**
- Input: `nums = [1,0,-1,0,-2,2], target = 0`
- Output: `[[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]`

**Example 2:**
- Input: `nums = [], target = 0`
- Output: `[]`

**Constraints:**
- `0 <= nums.length <= 200`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`
