# 3Sum

**Problem:** Given an array of n integers, find all unique triplets that sum to zero.

**Examples:**
- Input: `[-1,0,1,2,-1,-4]` → Output: `[[-1,-1,2],[-1,0,1]]`
- Input: `[]` → Output: `[]`
- Input: `[0]` → Output: `[]`

**Constraints:**
- Array length: 0 ≤ n ≤ 3000
- Element range: -10⁵ ≤ nums[i] ≤ 10⁵
- "Notice that the solution set must not contain duplicate triplets"
